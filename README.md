# Website_Login_Email_Capture_Business_Analyst
Overview

This project demonstrates the business analysis and requirements documentation process for a simple website login page designed to collect and manage customer email addresses.

The project was created to showcase skills in requirements gathering, process analysis, user story development, use case modeling, and documentation commonly performed by Business Analysts and Project Coordinators.

Files included:
-- Login_Screen_Business_Requirements.docx 
-- Login_Screen_Process_Flow.docx
-- Login_Screen_Use_Cases.docx
-- Login_Screen_User_Stories.docx
-- Login_Screen_Wireframes.docx
